Com uso de Flow

Ls:

Lista os arquivos e diretórios no diretório atual (em sistemas Unix/Linux).

Cd:

Permite a navegação entre diretórios. Por exemplo, cd pasta leva você para o diretório chamado "pasta".

Mkdir:

Cria um novo diretório. Por exemplo, mkdir novo_diretorio cria um diretório chamado "novo_diretorio".

Rm:

Remove arquivos ou diretórios. Use com cautela, pois os arquivos excluídos não vão para a lixeira/trash, eles são excluídos permanentemente.

Cp:

Copia arquivos ou diretórios. Por exemplo, cp arquivo.txt destino copia o arquivo "arquivo.txt" para o diretório "destino".

Mv:

Move ou renomeia arquivos ou diretórios. Por exemplo, mv arquivo.txt novo_nome.txt renomeia o arquivo "arquivo.txt" para "novo_nome.txt".

Grep:

Procura por padrões em arquivos. Por exemplo, grep "palavra" arquivo.txt busca a palavra "palavra" no arquivo "arquivo.txt".

Chmod:

Altera as permissões de acesso a arquivos ou diretórios. Por exemplo, chmod 755 arquivo.txt define permissões específicas para o arquivo.

Sudo:

Executa comandos com permissões de superusuário (administrador), útil para operações que exigem privilégios elevados.

Ping:

Verifica a conectividade com um host. Por exemplo, ping google.com verifica a conexão com o site do Google.
